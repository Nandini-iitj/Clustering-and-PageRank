Overview

This repository contains implementations for:

Clustering using k-center and k-means++
Web search using an inverted index
PageRank using Spark
Part 1: Clustering

Implemented Farthest First Traversal (k-center) and k-means++ algorithms.
Compared their performance using runtime and objective values.

Part 2: Web Search

Built an inverted index to support search queries on a collection of webpages.
Supports finding pages containing a word and positions of words in a page.

Part 3: PageRank

Implemented the PageRank algorithm using PySpark.
Computed node importance scores after multiple iterations.
